第4回｜タイムスタンプを追加して動画を再生可能にする、動画に合わせて歌詞をハイライト

目的

アラジンの挿入歌「Un Mundo Ideal」の歌詞を視覚的・機能的に表示し、語学学習ツールとしての完成度を高める。

作業内容

  1. 「Un Mundo Ideal」の歌詞を表示し、クリックで該当箇所へジャンプできるようにした。（▶️4, 19, 25, 33,46, 58, 68, 76, 87。▶️ボタンのクリックで、該当タイムスタンプにシークして再生する。タイムスタンプに合わせて、セリフをハイライト。各セリフのハイライトは、1秒前に開始し、次のタイムスタンプに到達するまで継続する。
  2. 再生中のタイミングに合わせて歌詞のハイライトが切り替わるようにJavaScriptを使用。
    処理の流れ
    • 各行に id="line1" などの識別子を付与
    • ハイライト対象を <span class="lyric-text"> で囲む
    • 歌詞と再生時間の対応をJavaScriptの配列で定義
    • YouTube IFrame APIを使って現在の再生時間を取得
    • Math.floor() で再生秒数を整数化し、対応する歌詞行を判断
    • 該当行の .lyric-text に .highlight クラスを追加
  3. 背景やフォントカラー、スタイルを整え、アラジンらしい幻想的な雰囲気にした


工夫したポイント

  • 動画と同期して歌詞が自動でハイライトされる仕組みを導入
  • 色使いやタイミングで「物語性」を演出し、楽しく学べるようにした

使用した技術・ツール

  • HTML / CSS / JavaScript
  • YouTube IFrame API
  • LyricsTranslate（歌詞参照サイト）
  • 生成AI（ChatGPT）
